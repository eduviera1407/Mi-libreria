# MyFavouriteBorder

Componente que combina un `IconButton` y un icono de favorito de Material UI.

## Props

- **color**: Color del icono (requerido).
- **disabled**: Estado deshabilitado del botón (opcional).
- **onClick**: Función que se ejecuta al hacer clic en el botón (opcional).

## Ejemplo

### Estado Predeterminado

<Canvas of={MyFavouriteBorder.Default} />

## Uso

```ts
import { MyFavouriteBorder } from 'milibreria';

function Example() {
    return (
        <MyFavouriteBorder
            color="red" 
            onClick={() => alert("¡Favorito añadido!")} 
        />
    );
}
export default Example;
